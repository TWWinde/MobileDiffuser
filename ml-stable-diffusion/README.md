# Vendored Core ML Stable Diffusion Package

This directory contains a local patched copy of Apple's `ml-stable-diffusion`
package used by MobileDiffuser.

The upstream README screenshots and example assets were removed from this
repository to keep the MobileDiffuser source tree focused and lightweight.

See the root `README.md` and `docs/` for MobileDiffuser-specific setup, model
resources, conversion steps, and iPhone deployment instructions.
